CoNote 对硬件的要求不高，但是推荐使用至少以下配置，以获得更好的体验。

## 推荐配置

- 内存：512MB及以上
- 硬盘：10GB及以上
- 处理器：1核及以上

## 软件依赖

使用CoNote时，会用到如下软件：

- Docker
- Docker Compose
- Browser
  - Chrome
  - Firefox
  - Edge
  - Safari
  - 其他现代浏览器

安装Docker，请参考[Docker官网](https://docs.docker.com/engine/install/)。Docker compose 已经内置在Docker软件中，不再需要单独安装。
